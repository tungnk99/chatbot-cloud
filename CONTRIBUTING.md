# Đóng Góp Cho Dự Án Chatbot Cloud

Cảm ơn bạn quan tâm đóng góp! Dự án tổ chức mã nguồn theo **Backend** và **Frontend**.

## Cấu Trúc Dự Án

- **Backend/**: Mã nguồn backend (chatbot service, tools)
  - `Backend/chatbot/`: Dịch vụ chatbot chính (Python)
  - `Backend/tools/`: Các microservice công cụ mở rộng (Python)
- **Frontend/**: Ứng dụng web (giao diện người dùng)

Khi thêm/sửa code, đặt đúng thư mục tương ứng (Backend hoặc Frontend).

## Quy Trình Đóng Góp

1. Fork repository
2. Tạo feature branch (`git checkout -b feature/AmazingFeature`)
3. Commit changes (`git commit -m 'Add some AmazingFeature'`)
4. Push to branch (`git push origin feature/AmazingFeature`)
5. Mở Pull Request

Chi tiết quy định code, API, Docker, Cloud Run: xem [AGENT.md](AGENT.md) và [docs/](docs/).
